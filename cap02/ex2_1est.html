Operador módulo e funções matemáticas

Digite números e clique em Mostrar para ver os resultados dos exemplos abaixo:


Módulo

Serve para obtermos o resto da divisão entre dois números.

Math.abs(num)

Retorna o valor absoluto de um número. Exemplo: -126. Ele retornará 126.

Math.ceil(num)

Arredonda um valor para cima. Se o número tiver decimais ele vai arredondar para o próximo inteiro.

Math.floor(num)

Arredonda um valor para baixo. Só retorna a parte inteira do número.

Math.pow(base, exp)

Retorna a base elevada ao expoente.

Math.random()

Retorna um número aleatório entre 0 e 1.

Math.round(num)

Arredonda um valor para o inteiro mais próximo.

Math.sqrt(num)

Retorna a raiz quadrada do número.
